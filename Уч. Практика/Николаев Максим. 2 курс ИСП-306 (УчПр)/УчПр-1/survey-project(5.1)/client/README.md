# Система опросов

Веб-приложение для создания и прохождения социологических опросов.

## Функциональность

### Для администраторов:
- Регистрация (требуется специальный код доступа)
- Создание новых опросов
- Добавление вопросов с 4 вариантами ответов
- Просмотр результатов опросов

### Для пользователей:
- Регистрация и вход
- Просмотр доступных опросов
- Прохождение опросов
- Просмотр своих результатов

## Установка и запуск

1. Скопируйте все файлы на веб-сервер
2. Откройте index.html в браузере
3. Для регистрации администратора используйте код: `admin123`

## Используемые технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage для хранения данных
- Адаптивный дизайн

## Структура проекта

- `/styles` - CSS файлы стилей
- `/scripts` - JavaScript файлы
- `index.html` - Главная страница
- `admin.html` - Панель администратора
- `user.html` - Панель пользователя
